# ExpenseTracker

## Run the project
- Clone the project.
- Run the following commands seqentially to run the project.
```bash
npm install
npm install @react-navigation/native @react-navigation/native-stack
npm install react-native-screens react-native-safe-area-context 
npx expo start
```
